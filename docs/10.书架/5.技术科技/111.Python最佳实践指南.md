这份人工编写的指南旨在为 Python 初学者和专家提供一个关于 Python 安装、配置、和日常使用的最佳实践手册。

这份指南是主观的，它与 Python'官方文档几乎，但不是完全不同。您在这不会找到每个 Python web 框架的列表。相反，您会发现一份优秀的简明列表，包含有强烈推荐的选项。
